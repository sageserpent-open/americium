  2. trait Configuration extends AnyRef

    Trait providing methods and classes used to configure property checks provided by the the forAll methods of trait GeneratorDrivenPropertyChecks (for ScalaTest-style property checks) and the check methods Checkers (for ScalaCheck-style property checks).

  3. class DiscardedEvaluationException extends RuntimeException

    Exception that indicates an evaluation of a property should be discarded, because a condition required by a property was not met by the passed values.

  4. class GeneratorDrivenPropertyCheckFailedException extends PropertyCheckFailedException
